# 🎲 Dice Chess Roller (Flask Version)

Minimal Flask app to roll chess-piece dice 🎯♟️  
Run it locally in Termux, Windows, Mac, wherever Python works.

## 🚀 How to Run

```bash
# clone it
git clone https://github.com/YOUR_USERNAME/dice-chess-flask.git
cd dice-chess-flask

# install Flask
pip install flask

# run it
python app.py
